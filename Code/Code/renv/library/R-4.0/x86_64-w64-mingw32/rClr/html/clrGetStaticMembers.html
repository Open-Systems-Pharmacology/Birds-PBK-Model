Description

Gets the static members for a type

Usage

clrGetStaticMembers(objOrType)

Arguments

objOrType

a CLR object, or type name, possibly namespace and assembly qualified type name, e.g. 'My.Namespace.MyClass,MyAssemblyName'.

Examples

## Not run: 
library(rClr)
cTypename <- "Rclr.TestCases"
clrGetStaticMembers(cTypename)
testClassName <- "Rclr.TestObject";
testObj <- clrNew(testClassName)
clrGetStaticMembers(testObj)
